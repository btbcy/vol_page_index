# install kcc
  - ```pip install --user KindleComicConverter```

# USAGE
1. input
   - path
   - name without .epub
   - remove [epub] ...
   - rename image inorder (from html)
   
2. kindle comic converter
   - Stretch/Upscale
   - change device profile if you need   
   - generate .cbz file and delete temp folder
   
3. copy .cbz file to your device
